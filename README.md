Flicker Mode
============

This mode only really looks good with frame blending enabled.

Technical
---------

* ANTIC mode F
* PRIOR swapped between $C0 (hue) and $40 (luma) every frame
* Pixel height expanded to 4 scan lines with VSCROL trick
